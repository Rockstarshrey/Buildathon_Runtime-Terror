Design a visually stunning, modern, and emotionally engaging mobile app UI for "Kisan Connect", a farmer-first platform.

The UI should feel alive, warm, and premium — not just simple. It should create a sense of trust, comfort, and pride for farmers using the app.

💎 Design Direction:
- Blend minimalism with visual richness
- Inspired by nature, sunrise over fields, soil textures, and fresh crops
- Use soft gradients, depth, and smooth shadows
- Avoid flat boring design — add subtle visual layers

🎨 Advanced Color System:
- Primary: Deep green (#2E7D32)
- Gradient: Green → light green (#2E7D32 → #66BB6A)
- Background: Warm cream (#F9FBE7) with slight texture feel
- Accent: Golden yellow (#FBC02D)
- Secondary accent: Sky blue (#81D4FA) for freshness
- Use gradient buttons instead of flat colors

✨ Visual Enhancements:
- Soft gradient cards (not plain white)
- Glassmorphism or slight blur effects for cards
- Floating UI elements (cards slightly lifted)
- Smooth shadows with depth
- Use modern illustrations of farmers, crops, tractors

🏠 Home Screen (Make it PREMIUM):
- Top curved header with gradient background (green tones)
- Greeting with avatar + “Good Morning 🌿”
- Weather card with icon + subtle animation (sun/cloud)
- Quick action cards with icons + slight 3D feel:
   - Sell Crops 🌾
   - Buy Products 🛒
   - Market Prices 📊
   - Community 🤝

- Add a horizontal scroll section:
   👉 “🔥 High Demand Crops Today”
   (with attractive cards + images)

- Add a “Daily Tip” card with icon + highlight color

🛒 Selling Experience (Make it SMART):
- Step-by-step animated flow
- Each step slides smoothly
- Add image preview with rounded corners
- Use progress indicator (Step 1 → Step 2 → Step 3)
- Add satisfying success animation after posting

💬 Chat UI (Modern + Attractive):
- Use gradient chat bubbles
- Rounded large bubbles
- Voice message UI with waveform animation
- Profile avatars with colored rings

📊 Market Prices (Make it Beautiful):
- Use card-based crop display
- Add mini graphs inside cards
- Highlight best price with glowing badge ✨
- Use color-coded trends (green/red)
- Smooth animated graph transitions

🎯 Micro-Interactions (VERY IMPORTANT):
- Button press ripple effect
- Card hover/press lift animation
- Smooth page transitions
- Subtle loading animations (not boring spinners)

🌍 Accessibility + Delight:
- Large fonts but stylish typography
- Multi-language toggle
- Voice-first option
- Use icons + visuals more than text

❤️ Emotional Layer:
- Add friendly messages:
   “Great! Your crop is now live 🌾”
   “Good price today! Sell now 💰”
- Make the app feel like a companion, not a tool

🎯 Final Goal:
Create a UI that is not just usable — but delightful, visually rich, and addictive — so farmers feel proud using it and naturally spend more time exploring it.
