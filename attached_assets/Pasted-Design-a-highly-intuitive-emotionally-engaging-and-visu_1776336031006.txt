Design a highly intuitive, emotionally engaging, and visually clean mobile app UI for "Kisan Connect", a farmer-focused platform that enables selling crops, buying agricultural products, viewing market prices, and connecting with other farmers.

The UI should prioritize simplicity, trust, and ease of use for semi-digital users. The design must feel calm, natural, and friendly — inspired by farmland, soil, and greenery.

🎨 Visual Style:
- Soft, nature-inspired color palette
- Primary green (#2E7D32) for trust and action
- Light green (#A5D6A7) for highlights
- Warm cream background (#F9FBE7)
- Yellow accent (#FBC02D) for CTAs
- Dark grey text (#263238)
- Avoid harsh contrast or overly bright/dark themes

🧩 Layout Principles:
- Card-based UI with soft shadows
- Rounded corners (16–20px)
- Clear visual hierarchy
- Generous spacing (breathing room)
- Large touch-friendly components

🏠 Home Screen Experience:
- Personalized greeting (e.g., "Welcome back, Farmer 🌿")
- Weather + crop advisory card
- Quick action grid (Sell Crops, Buy Inputs, Market Prices, Community)
- Daily farming tip banner (dynamic content)
- "What’s trending nearby" section (local demand crops)

🛒 Selling Flow (Ultra-simple):
- Step-by-step guided flow (not a long form)
- Use icons + minimal text
- Option for voice input 🎤
- Smart suggestions (auto-fill crop names)
- Visual confirmation after posting

💬 Community & Chat:
- Familiar messaging UI (WhatsApp-inspired)
- Support voice notes (important for low-literacy users)
- Highlight nearby farmers first
- Add "Ask Expert" feature

📊 Market Prices:
- Show only essential data
- Use simple graphs (no complex charts)
- Highlight best selling price nearby
- Use color indicators (green = good price)

🧠 UX Psychology (VERY IMPORTANT):
- Reduce cognitive load (no clutter)
- Use progressive disclosure (show step-by-step)
- Reward actions (small success messages like "Great! Your crop is live 🌾")
- Add micro-interactions (button animations, soft transitions)

🌍 Accessibility:
- Large readable fonts
- High visibility in sunlight
- Multi-language support (regional languages)
- Voice-first approach where possible

🎯 Goal:
Design an experience that feels effortless, friendly, and trustworthy — so farmers feel comfortable using the app daily and spending more time exploring features.
